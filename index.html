><><>MCBash<><><> == === What is MCBash? === MCBash is a set of bash utilities that allows for ease-of-mind for server administrators, by way of automated integrity checks, automated server and MySQL backups, and automated server.log deletion.

=== How does MCBash work? === MCBash is a set of scripts that are automatically executed by //Cron//, a scheduling utility, at desired intervals.

=== How often does MCBash back up my stuff? === However often you want it to. However, the design from the begenning was this:

  • MCBash runs your startup script at boot. ** MCBash checks every 5 minutes to ensure that your startup script is still running. If it's not, it will restart it.
  • MCBash clears your server.log, backs up, and compresses your entire server directory every hour. ** MCBash compresses all of your hourly backups at the end of the day into one compressed file.
